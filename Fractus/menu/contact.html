## 작품에 문제가 발생할 경우
* 웹 페이지가 제대로 작동하지 않을 경우, 기존의 웹 페이지를 닫고 새로운 웹페이지 창을 열고 주소창에 "localhost:9000"를 입력하고 기다려주세요. 이때 페이지가 제대로 작동하지 않는다면 새로고침('F5')을 해주세요.
* 아두이노가 제대로 작동하지 않을 경우, 아두이노 전원(usb 포트 옆 빨간 버튼) 눌러 다시 실행해주세요.
* 프로그램이 제대로 작동하지 않을 경우, visual studio code 프로그램 창의 터미널에 지정된 파일 경로를 "server" 폴더로 지정해주고, "node index.js"를 입력해주세요. 기울기 데이터가 출력되기 시작하면, 새로운 웹 페이지 창을 열어주세요. 웹 페이지 주소창에는 "localhost:9000"를 입력해주세요.
* 웹 페이지에 재접속하고도 페이지가 제대로 뜨지 않을 경우, 새로고침(F5)을 눌러주세요.
* 노트북의 배터리가 없는 경우, 작가에게 알려주세요.
* 위와 같은 조치를 취하여도 문제가 해결되지 않는다면 아래의 SNS 계정으로 연락해주세요.
* 작품의 문제 외에도 작품과 관련하여 언제든지 편하게 연락을 주셔도 됩니다.
